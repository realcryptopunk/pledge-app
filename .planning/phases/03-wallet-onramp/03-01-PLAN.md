<task type="auto">
  <name>Task 2: Create PrivyManager singleton service</name>
  <files>Core/Services/PrivyManager.swift</files>
  <action>
Create PrivyManager as a @MainActor ObservableObject singleton following the established pattern (like HealthKitManager.shared).

Key implementation:
1. `static let shared = PrivyManager()`
2. Store the single `Privy` instance as a private property
3. @Published properties: `isAuthenticated: Bool`, `walletAddress: String?`, `userPhone: String?`, `authError: String?`, `isLoading: Bool`
4. `initialize()` method — call `PrivySdk.initialize(config:)` with appId/appClientId from a config (use placeholder strings "YOUR_PRIVY_APP_ID" and "YOUR_PRIVY_APP_CLIENT_ID" with a TODO comment — user will fill from Privy Dashboard)
5. `sendOTP(to phoneNumber: String) async throws` — calls `privy.sms.sendCode(to:)` with E.164 format
6. `verifyOTP(code: String, phoneNumber: String) async throws` — calls `privy.sms.loginWithCode(_:sentTo:)`, then automatically calls `createWalletIfNeeded()`
7. `createWalletIfNeeded() async throws` — checks if user has embedded ethereum wallets, if not calls `user.createEthereumWallet()`. Stores address in `walletAddress`.
8. `checkAuthState() async` — calls `privy.getAuthState()`, updates published properties. If authenticated, also loads existing wallet address.
9. `signOut() async` — calls `privy.logout()`, clears all published state.

Use `import PrivySDK` at top. The Privy instance type is `Privy` (protocol). Create via `PrivySdk.initialize(config:)`.

Do NOT wire into AppState yet — that's Plan 02's job. This is a standalone service.
  </action>
