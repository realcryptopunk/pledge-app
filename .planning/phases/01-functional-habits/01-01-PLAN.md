<task type="auto">
  <name>Task 2: Create HealthKitManager service</name>
  <files>Core/Services/HealthKitManager.swift</files>
  <action>
  Create `Core/Services/HealthKitManager.swift` with a @MainActor ObservableObject class:

  ```swift
  import HealthKit
  ```

  **Properties:**
  - `let healthStore = HKHealthStore()`
  - `@Published var isAuthorized = false`
  - `static let shared = HealthKitManager()` (singleton for easy access)

  **Authorization method** `requestAuthorization()`:
  - Check `HKHealthStore.isHealthDataAvailable()` first
  - Define read types: stepCount, sleepAnalysis, appleExerciseTime (or workoutType)
  - Request authorization with `healthStore.requestAuthorization(toShare: [], read: readTypes)`
  - Set `isAuthorized = true` on success
  - Use async/await (iOS 17+ target)

  **Query methods (all async throws):**

  1. `func fetchSteps(for date: Date) async throws -> Double`
     - Query HKQuantityType(.stepCount) for the given date (midnight to midnight)
     - Use HKStatisticsQuery with .cumulativeSum
     - Return total step count as Double
     - Wrap in withCheckedThrowingContinuation for the callback-based API

  2. `func fetchSleepHours(for date: Date) async throws -> Double`
     - Query HKCategoryType(.sleepAnalysis) for the night before (8pm previous day to noon current day)
     - Filter for .asleepUnspecified, .asleepCore, .asleepDeep, .asleepREM (not .inBed)
     - Sum total sleep duration in hours
     - Use HKSampleQuery, calculate duration from each sample's startDate/endDate

  3. `func fetchWorkoutMinutes(for date: Date) async throws -> Double`
     - Query HKQuantityType(.appleExerciseTime) for the given date
     - Use HKStatisticsQuery with .cumulativeSum
     - Return total minutes as Double

  **Important:** Use `withCheckedThrowingContinuation` to bridge callback-based HKQuery to async/await. Do NOT use the older completion handler pattern directly in async functions.

  **Error handling:** Create a simple HealthKitError enum with cases: .notAvailable, .authorizationFailed, .queryFailed(Error)
  </action>
