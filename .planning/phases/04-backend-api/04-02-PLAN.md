  <name>Task 1: Create Coinbase session token edge function</name>
  <files>supabase/functions/coinbase-session-token/index.ts</files>
  <action>
Create edge function at `supabase/functions/coinbase-session-token/index.ts`.

The Coinbase Onramp session token flow:
1. Receive POST with `{ wallet_address, amount }` from iOS app
2. Create a JWT signed with the CDP API key secret (ES256 algorithm):
   - Header: `{ alg: "ES256", kid: CDP_API_KEY_NAME, typ: "JWT", nonce: random_hex }`
   - Payload: `{ sub: CDP_API_KEY_NAME, iss: "coinbase-cloud", aud: ["retail_rest_api_proxy"], nbf: now, exp: now + 5min, uris: [] }`
3. POST to `https://api.developer.coinbase.com/onramp/v1/token` with:
   - Header: `Authorization: Bearer <JWT>`
   - Body: `{ destination_wallets: [{ address: wallet_address, blockchains: ["base"] }] }`
4. Return the `token` from the response

Environment variables needed (set via `supabase secrets set`):
- `CDP_API_KEY_NAME` — Coinbase Developer Platform API key name
- `CDP_API_KEY_SECRET` — Coinbase Developer Platform API key secret (PEM format EC private key)

Use the Web Crypto API or `jose` library (available in Deno) for JWT signing with ES256. Import jose from `https://deno.land/x/jose/index.ts` or npm specifier.

CORS headers: Allow requests from the iOS app (use `*` for development).

Error handling: Return appropriate HTTP status codes (400 for missing params, 500 for Coinbase API errors, 401 for auth failures). Include error message in response body for debugging.

Deploy: `supabase functions deploy coinbase-session-token`
Set secrets: `supabase secrets set CDP_API_KEY_NAME=<key> CDP_API_KEY_SECRET=<secret>`

Create a helper function (either on CoinbaseOnrampView or as a separate service) to fetch the session token:
```swift
static func fetchSessionToken(walletAddress: String, amount: Double) async throws -> String {
    let url = URL(string: "\(EnvConfig.supabaseURL)/functions/v1/coinbase-session-token")!
    var request = URLRequest(url: url)
    request.httpMethod = "POST"
    request.setValue("Bearer \(EnvConfig.supabaseAnonKey)", forHTTPHeaderField: "Authorization")
    request.setValue("application/json", forHTTPHeaderField: "Content-Type")

    let body = ["wallet_address": walletAddress, "amount": amount] as [String: Any]
    request.httpBody = try JSONSerialization.data(withJSONObject: body)

    let (data, response) = try await URLSession.shared.data(for: request)
    // Parse token from response JSON
    let json = try JSONSerialization.jsonObject(with: data) as? [String: Any]
    guard let token = json?["token"] as? String else {
        throw NSError(domain: "CoinbaseOnramp", code: -1, userInfo: [NSLocalizedDescriptionKey: "Missing session token"])
    }
    return token
}
```
